Colours

Colour Palette

In my design system, I've carefully curated a palette of colors to evoke the essence of the Outback Nursery brand and provide a visually harmonious experience for the site users. Let's explore the hues that is going to define the digital identity for Outback Nursery.

Primary Palette

Class name: u-text-primary, u-text-primary-as-bg
Hex code: #605F4F
Uses: Body text
Class name: u-bg-primary
Hex code: #E6FEF7
Uses: Section background and CTAs
Class name: u-bg-primary-light, u-text-primary-light
Hex code: #F4FFFF
Uses: Section background and Text colour for CTAs and Notice section
Class name: u-bg-primary-dark
Hex code: #B5F6EB
Uses: Background for specific content (May be a Testimonial card)

Secondary & Accent Palette

Class name: u-text-secondary, u-accent
Hex code: #BA2D0B
Uses: CTA, menu and links
Class name: u-bg-secondary
Hex code: #FDDB91
Uses: Footer section background
Class name: u-bg-accent-light
Hex code: #FF9E77
Uses: Decorative elements (Borders or dividers to create visual separation)
Class name: u-bg-accent-dark
Hex code: #7A1700
Uses: Notice section, Social media icons
