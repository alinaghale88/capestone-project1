Flexible Layout Classes

The Flexible Layout Classes section of my design system outlines a set of classes tailored to create flexible and responsive layouts using flexbox.

Class Name Property Value
.l-flex display flex
.l-direction-row flex-direction row
.l-direction-row-dt flex-direction row
.l-direction-column flex-direction column
.l-wrap flex-wrap wrap
.l-justify-flex-end justify-cotent flex-end

Container classes

The Container classes section of our design system encompasses various classes tailored to manage the layout and width of containers across different screen sizes.

Here are the available classes and their corresponding padding values:

Class Name Property Value
.l-full-width width 100%
.l-desktop-full-width width calc(100% - var(--side-nav-width))
.l-max-width-mobile max-width 480px
.l-max-width-desktop max-width 1170px
.l-text-wrapper-desktop max-width 760px
